== Description ==

Creates Microlink cards effortless. Three ways of using this plugin:
* Add `class="microlink"` to the desired links from the text editor (default markup will be used)
* Use a shortcode like [microlink href="https://microlink.io/"] to generate the link for you
* Use the editor Microlink button to open the creator

== Installation ==

Just activate the plugin, nothing else to do.

If you do have an API key, go to **Settings** -> _Microlink_
